# LARAVEL RULE NOTEBOOK

<br>

## Before Deployeing Laravel application collect Requirement:

1. Laravel version eg. `Laravel 10, Laravel 11, Laravel 12`.
2. PHP Version eg. `php 8.1, php 8.2, php 8.3, php 8.4`.
3. Composer Version eg. `2.9.5`.
4. MySQL version eg. `MySql 8.0, MySql 8.4, MySql 9.6.0`.



## Website Images

### 1. Laravel Logo
* PNG (transparent)
* 200–300px width
* Keep logo under 100 KB max

### 2. Favicon
* PNG (transparent)
* 48×48 → 5–15 KB

### 3. Hero Banner (1920×1080)
* WebP: 100–300 KB
* JPG: 200–500 KB

### 4. Feature Image (800×600)
* WebP: 50–150 KB
* JPG: 80–250 KB

### 5. Thumbnail (300×300)
* WebP: 10–40 KB
* JPG/PNG: 20–70 KB

### 6. Blog / Social Share (1200×630)
* WebP: 80–200 KB
* JPG: 150–350 KB


## Important performance base packages for Laravel Application

1. nginx
2. PHP-FPM
3. MySQL
4. supervisor
5. Redis (optional but recommended for queues)
6. OPcache
